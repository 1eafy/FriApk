# FriApk
> 基于Frida、androguard，联动安卓模拟器脱壳，进行静态、动态漏洞检测工具，非法获取用户设备、敏感信息行为检测（猜想）

### requirements.txt
```
networkx == 2.5
asn1crypto == 1.4.0
frida == 12.11.18
```

###### **注: frida版本与模拟器的frida-server版本尽量需要对应上**
